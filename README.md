# Chocapix - Front end

## Development
Run:
```
npm install -g grunt bower

npm install
bower install
grunt serve
```

Default backend logins: `admin:admin` `ntag:ntag` `tizot:tizot`

## Build for production
```
npm install -g gulp
gulp build
```
